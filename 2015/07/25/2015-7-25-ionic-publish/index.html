publish to firm.im

share the app url make more people can see it.

for android
1. get anroid apk file in $projectname/platforms/android/build/outputs/apk/anroid-debug.apk
2. upload the apk to http://firm.im
for ios
1. open the $prodjectname.xcodeproj wiht xcode
2. connect your device and build a app in prodcut menu
3. archive the project and generate the $prodjectname.xcarchive folder
4. export the $prodjectname.xcarchive to a .ipa file
5. upload the $prodjectname.ipa file to http://firm.im
