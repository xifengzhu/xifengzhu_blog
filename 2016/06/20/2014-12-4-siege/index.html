Web压力测试工具Siege介绍

Siege是一款不错的网站压力测试工具，它可以非常容易地定制并发访问人数以及并发时间，安装、使用也非常方便。
在mac上你只需要运行:

brew install Siege

就可以在你的电脑上快速的安装好。
安装好之后你就可以通过输入: man siege 来查看帮助文档，这里就不一一赘述了。
直接来对一个网站进行测试：

siege http://xifengzhu.github.io/xifengzhu_blog/ -c200 -t5

输入上面的命令就，启动200个用户（线程）在5分钟内对某一页面http://xifengzhu.github.io/xifengzhu_blog/进行并发访问

如果你想对网站的多个页面进行并发的话, 新建一个文件urls.txt，里面的内容为（只是例子，任何url都可以)：

http://xifengzhu.github.io/xifengzhu_blog/
http://xifengzhu.github.io/xifengzhu_blog/ruby/2014/11/09/mechanize.html

然后启动siege进行压力测试

siege -f urls.txt -c200 -t5

并发测试完毕后，会打印出一些结果信息：

Lifting the server siege..      done.

Transactions:                 78 hits
Availability:              97.50 %
Elapsed time:              83.48 secs
Data transferred:           6.23 MB
Response time:             36.40 secs
Transaction rate:           0.93 trans/sec
Throughput:             0.07 MB/sec
Concurrency:               34.01
Successful transactions:          78
Failed transactions:               2
Longest transaction:           68.95
Shortest transaction:           0.00

如果你的网站需要登录怎可以用带上header参数的方式解决：

(1)登录你要测试的网站
(2)打开调试工具, 将remember_user_token"your remember_user_token_token"(其他的cookie键值也可以)
(3)siege -c 30 -r 10 --header="Cookie: remember_user_token=your remember_user_token" -f urls.txt
