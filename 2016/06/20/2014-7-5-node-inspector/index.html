node inspector 用法介绍

1、安装node-inspector

2、测试是否安装成功？

3、进行debug调试。

1）运行node 调试模式

node提供了两种debug模式运行：

node —debug[=port] filename （这种方式，其实就是在指定的端口（默认为 5858）监听远程调试连接）
node —debug-brk[=port] filename （这种方式在监听的同时，会在代码执行的时候，强制断点在第一行，这样有个 好处就是：可以debug到node内部的是如何运行的）


这里采用第二种方式运行：
node —debug-brk  app.js

2)启动 node-inspector

你可以在浏览器中注输入http://127.0.0.1:8080/debug?port=5858就可以开始。。。。。。
