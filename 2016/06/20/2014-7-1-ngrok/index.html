ngrok

  • 发布 2016-06-20
  • 更新 2016-06-20
  • 阅读

ngork 是一个将你本地端口映射成一个可以从外网方的地址

//如果你本地有一个rails的server在启动，端口为3000.则你可以运行 ./ngrok 3001

//则nrok会映射一个地址给你 http://3a4bfceb.ngrok.com -> 127.0.0.1:3001

更多配置请访问https://ngrok.com/usage

简单的运行

// 代理你电脑3001端口， 运行后通过http://127.0.0.1:4040查看请求状态
ngrok 3001
验证运行

// authtoken你在官方注册，就会生成一个, authtoken只要在电脑上运行一次就可以了,

// 如果你验证了的话ngrok 生成的前缀域名不会被占用和变化
ngrok -authtoken U54UzZiDLFQxyZ-Ow*** 3001

需要别人验证才可以浏览

// helmet用户名, 12345密码

ngrok -httpauth="helmet:12345" 3001

设置前缀域名

ngrok -subdomain=example 3001

tcp代理

ngrok -proto=tcp 22

指定其它服务器代理

//为192.168.1.16服务器的3001端口代理

ngrok 192.168.1.16:3001

运行多个类型服务

// ngrok.yml配置类型如下

tunnels:
  client:
    proto:
      https: "3000"
    auth: user:password
  c99bba1.ngrok.com:
    proto:
      http: "3000"
  ssh:
   proto:
      tcp: "22"

// config: 配置文件路径

// start: 运行哪几种类型服务(client, ssh c99bba1.ngrok.com)

ngrok -log=stdout -authtoken=U54UzZiDLFQxyZ-Ow*** -config=ngrok.yml start client ssh c99bba1.ngrok.
